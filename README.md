## Run Locally

Clone the project

```bash
  git clone https://github.com/mohammad-naimur-rahman/react-components-market.git
```

Go to the project directory

```bash
  cd react-components-market
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```
